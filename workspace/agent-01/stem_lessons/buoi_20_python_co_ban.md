## Hoạt Động Thực Hành (65 phút)

### Hoạt động 1: Chạy Python đầu tiên (20 phút)
**Các bước:**
1. Mở trình duyệt, vào replit.com
2. Tạo tài khoản (miễn phí)
3. Tạo "Repl" mới → chọn Python
4. Gõ lệnh đầu tiên:

```python
print("Xin chào, thế giới!")
```

5. Bấm nút "Run" → xem kết quả

### Hoạt động 2: Biến và in ấn (25 phút)
**Khái niệm biến:**
```python
# Tạo biến
ten = "Minh"
tuoi = 10

# In ra màn hình
print("Tên:", ten)
print("Tuổi:", tuoi)
```

**Thực hành:**
```python
# Nhập tên và in chào
ten = input("Nhập tên của bạn: ")
print("Xin chào", ten)

# Tính tổng hai số
a = 5
b = 3
tong = a + b
print("Tổng:", tong)
```

### Hoạt động 3: Tính toán đơn giản (20 phút)
**Các phép toán:**
```python
# Cộng, trừ, nhân, chia
a = 10
b = 3

print(a + b)  # 13
print(a - b)  # 7
print(a * b)  # 30
print(a / b)  # 3.333...

# Chia lấy dư
print(a % b)  # 1

# Luỹ thừa
print(a ** b) # 1000
```

**Bài tập:**
- Tính chu vi và diện tích hình chữ nhật
- Tính điểm trung bình 3 môn
