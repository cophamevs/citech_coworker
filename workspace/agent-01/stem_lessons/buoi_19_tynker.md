# Buổi 19: Tynker - Lập Trình Sáng Tạo

## Thông Tin Buổi Học
- **Độ tuổi**: 8-14 tuổi
- **Thời lượng**: 90 phút
- **Lĩnh vực**: Công nghệ (Lập trình)

## Mục Tiêu Buổi Học
1. Làm quen với Tynker
2. Tạo game và ứng dụng đơn giản
3. Khám phá các tính năng khác nhau

## Nội Dung Lý Thuyết (15 phút)

### Tynker là gì?
- Nền tảng lập trình giáo dục
- Tương tự Scratch
- Có cả phiên bản miễn phí và trả phí
- Website: tynker.com

### Đặc điểm:
- Khối kéo thả trực quan
- Nhiều template game có sẵn
- Hỗ trợ Minecraft modding
- Có thể tạo ứng dụng di động

### So sánh:
| Scratch | Tynker |
|---------|--------|
| Miễn phí hoàn toàn | Có phiên bản trả phí |
| Cộng đồng lớn | Nhiều template game |
| Đơn giản | Nhiều tính năng |

## Hoạt Động Thực Hành (65 phút)

### Hoạt động 1: Đăng ký và khám phá (15 phút)
**Các bước:**
1. Vào tynker.com
2. Đăng ký tài khoản (miễn phí)
3. Khám phá các mục:
   - Puzzles: Câu đố lập trình
   - Coding: Tạo dự án
   - Games: Chơi game mẫu

### Hoạt động 2: Làm game Flappy Bird (30 phút)
**Mục tiêu:** Tạo game Flappy Bird đơn giản

**Các bước:**
1. Chọn template "Flappy Bird"
2. Tùy chỉnh nhân vật
3. Lập trình:
   - Chim rơi xuống khi không bấm
   - Chim bay lên khi bấm
   - Ống di chuyển sang trái
   - Khi chạm ống hoặc đất → Game Over

**Tùy chỉnh:**
- Đổi hình chim thành khác
- Thay đổi tốc độ
- Thêm âm thanh

### Hoạt động 3: Tạo câu đố (20 phút)
**Mục tiêu:** Tạo câu đố trắc nghiệm

**Các bước:**
1. Tạo dự án mới "Câu đố"
2. Thêm câu hỏi và đáp án
3. Lập trình kiểm tra đúng/sai
4. Hiển thị kết quả

## Bài Tập Về Nhà
- Hoàn thành game Flappy Bird
- Tạo một câu đố cho gia đình
