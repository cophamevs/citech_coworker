# Buổi 29: micro:bit - Máy Tính Nhỏ Cho Trẻ

- **Độ tuổi**: 10-14 tuổi
- **Thời lượng**: 90 phút
- **Lĩnh vực**: Công nghệ (Hardware/Programming)

## Mục Tiêu Buổi Học
1. Hiểu về micro:bit
2. Lập trình cho micro:bit
3. Tạo ứng dụng với các cảm biến

## Nội Dung Lý Thuyết (15 phút)

### micro:bit là gì?
- Máy tính nhỏ do BBC phát triển
- Có màn hình LED, nút bấm, cảm biến
- Lập trình bằng khối (Block) hoặc Python
- Giá rẻ, phù hợp giáo dục

### Các tính năng:
- 25 đèn LED đỏ
- 2 nút bấm (A, B)
- Cảm biến gia tốc
- Bluetooth
- Loa (phiên bản mới)

### Lập trình:
- Website: makecode.microbit.org
- Kéo thả khối như Scratch

## Hoạt Động Thực Hành (65 phút)

### Hoạt động 1: Xin chào micro:bit (20 phút)
**Các bước:**
1. Vào makecode.microbit.org
2. Tạo project mới
3. Kéo khối "hiển thị" → vẽ hình trái tim
4. Tải về và nạp vào micro:bit

**Code mẫu:**
```
khi bấm nút A
hiển thị trái tim
```

### Hoạt động 2: Hiển thị chữ chạy (25 phút)
**Code:**
```
khi bấm nút A
hiển thị chữ "HELLO"
```

**Nâng cao:**
```
khi bấm nút A
hiển thị chữ "HELLO"
đợi 500 ms
hiển thị chữ "HI!"
```

### Hoạt động 3: Cảm biến gia tốc (20 phút)
```
khi lắc
hiển thị icon "vui"
chơi nhạc melody ba-ding
```
